# 2026年夏 家族4人関西旅行

## 旅行の概要

2026年8月20日から25日まで、家族4人で岡山、大阪、京都、USJを巡る5泊6日の旅行です。

8月20日は、晴が東京方面から、家族3人が沖縄から関西空港経由で移動します。新大阪で4人が合流して岡山へ向かい、その後は大阪を拠点に観光します。旅行後半はUSJ周辺へ宿泊先を移し、8月24日に家族でUSJを一日楽しみます。8月25日は家族4人で沖縄へ帰ります。

## 旅の大きな流れ

- 8月20日: 東京組と沖縄組が新大阪で合流し、岡山へ移動
- 8月20日から23日: 淀屋橋周辺に3泊し、大阪と京都を観光
- 8月22日: 京都観光とリアル脱出ゲーム
- 8月23日: USJ周辺へ移動して2泊
- 8月24日: USJを一日かけて楽しむ
- 8月25日: 関西空港から家族4人で沖縄へ帰る

## この旅行で大切にすること

- 交通の確定情報と、相談中の候補を混同しない
- 観光を詰め込みすぎず、暑さと疲労に応じて休憩する
- USJ、食事、京都の候補は家族全員がWeb上で追記する
- 変更した内容はSupabaseを通して別端末へ同期する
- 予約番号、QRコード、電話番号などの機微情報は公開サイトに保存しない

## 共同編集する内容

### USJ作戦

朝一番に向かうエリア、優先アトラクション、昼食時間、休憩、混雑時の切替条件を家族で共有します。

### 食事候補

京都、大阪、USJ周辺で行きたい店を、エリア、予算、予約要否、行きたい理由と一緒に記録します。

### 京都メモ

行きたい場所、抹茶や和菓子、暑さ対策、移動量を考慮して、無理のない一日を組み立てます。

### 共有メモ

家族への連絡、当日確認したいこと、新しく見つけた候補をフィード形式で投稿します。投稿後も編集と削除が可能です。
